# `data/car2024.xlsx` — 자동차보험 진료건수

한의원·정형외 등 **교통사고 후유증·통증** 수요를 볼 때 참고할 수 있도록, 시·군·구별 자동차보험 진료건수를 넣습니다.

## 필수 형식

- **첫 시트**에 표 형식.
- 지역 컬럼: 이름에 `시군구`, `지역`, `구역` 등이 포함되면 자동 인식. 없으면 **첫 번째 열**을 지역으로 사용.
- 건수 컬럼: 이름에 `건수`, `진료`, `합계` 등이 포함된 숫자 열을 자동 선택.
- **시도 + 시군구**가 나뉘어 있으면 두 열을 합쳐 `서울특별시 강남구` 형태로 맞춥니다.

## 마스터와 표기 매칭

`bluedot_master`의 행정동 문자열에서 앞 두 단어(예: `강원 강릉시`)를 시·군·구 키로 쓰므로, 엑셀의 `시군구` 값도 가능한 한 **동일한 표기**로 맞추세요.

## 순위 의미

- **전국 시·군·구 합계 대비 %**: 엑셀에 있는 모든 시·군·구의 건수 합을 전국 대비 분모로 사용합니다.
- **순위**: 건수 내림차순 1위 = 최다.

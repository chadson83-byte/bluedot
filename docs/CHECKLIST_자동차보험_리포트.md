# 자동차보험 진료건수 배너가 안 보일 때 — 할 일 전부

## 1. 어디에 표시되나?

- **지도·목록 화면이 아니라**, **「정밀 컨설팅 리포트」 모달을 연 뒤**  
  제목(`개원 입지 타당성…`) 바로 아래 **파란/노란 배너**에 나옵니다.
- 분석을 한 번 돌린 뒤, 순위 카드의 **Rank** 또는 **정밀 컨설팅 리포트** 버튼을 눌러야 합니다.

## 2. 백엔드(필수)

| # | 작업 |
|---|------|
| 1 | **FastAPI 서버 실행** — `uvicorn main:app --reload` 등으로 `http://127.0.0.1:8000` 에 떠 있어야 합니다. |
| 2 | **코드 저장 후 서버 재시작** — `main.py` / `engine/` 수정 뒤에는 재시작(또는 `--reload`로 자동 반영). |
| 3 | **`openpyxl` 설치** — `pip install openpyxl` 또는 `pip install -r requirements.txt` (엑셀 읽기용). |
| 4 | **`data/car2024.xlsx` 파일** — 프로젝트 루트 기준 `data/car2024.xlsx` 경로에 있어야 합니다. (없으면 노란 안내 문구만 표시) |

## 3. 프론트·연결

| # | 작업 |
|---|------|
| 1 | **`index.html`의 `BLUEDOT_API_BASE`** — 실제 API 주소와 같아야 합니다. (기본 `http://127.0.0.1:8000`) |
| 2 | **강력 새로고침** — `Ctrl+F5` 또는 캐시 비우기로 `app.js` 구버전이 남지 않게 합니다. |
| 3 | **`file://` 로 HTML만 열면** CORS/혼합 콘텐츠 이슈가 날 수 있으니, 가능하면 **로컬 서버로 페이지 제공**을 권장합니다. |
